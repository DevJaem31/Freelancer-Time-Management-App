<h1 align="center">Freelancers-Time-Management-App</h1>

A MERN-stack productivity app designed for freelancers to manage time, clients, projects, tasks, invoices, and budgeting â€” all in one place.

## Tech Stack

- Frontend: React, Tailwind CSS, React Router
- Backend: Node.js, Express
- Database: MongoDB (Mongoose)
- Auth: Google OAuth + JWT
- Deploy: Vercel (Frontend), Render (Backend)

## Features (MVP)

- Google Login
- Projects & Tasks Management
- Time Tracker + Daily Log
- Time-block Calendar
- Client Directory
- Invoice Generator (PDF)
- Budgeting & Expenses
- Responsive dashboard UI
